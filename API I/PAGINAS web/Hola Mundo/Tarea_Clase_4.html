ISO/IEC 25010

Adecuación Funcional

Representa la capacidad del producto software para proporcionar funciones que satisfacen las necesidades declaradas e implícitas de los usuarios cuando el producto se usa en las condiciones especificadas. Esta característica se subdivide a su vez en las siguientes subcaracterísticas:

  • Completitud funcional: Grado en el que el conjunto de funcionalidades del producto cubre todas las tareas y los objetivos de usuario especificados.
  • Corrección funcional: Capacidad del producto o sistema para proveer resultados exactos cuando es usado por los usuarios especificados.
  • Pertinencia funcional: Capacidad del producto software para proporcionar un conjunto de funciones que facilitan la consecución de tareas y objetivos de usuario especificados.
